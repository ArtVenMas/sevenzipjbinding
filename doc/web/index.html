What's this all about?

7-Zip-JBinding is a java wrapper for 7-Zip C++ library. It allows extraction of many archive formats using very fast native library directly from java through JNI. Supported archive types:
  • Extraction of
    7-ZIPZIPRARTARSPLITLZMAISOHFSGZIPCPIOBZIP2ZARJ
  • 12 example code snippets
  • Over 3000 JUnit tests
7-ZIP, ZIP, RAR, TAR, SPLIT, LZMA, ISO, HFS, GZIP, CPIO, BZIP2, Z, ARJ
7-ZIPZIPRARTARSPLITLZMAISOHFSGZIPCPIOBZIP2ZARJ
The archive creation and some other archive types (such as RPM, CDEB, NSIS) are in the planning stage.

The library contains more that 2000 JUnit test, that automaticly checks 7-Zip-JBinding in every possible way.

How does it work?

The 7-Zip-JBinding consists of two parts: the java part and the native part. The java part introduces a cross-platform java interface of 7-Zip library. The native part of 7-Zip-JBinding communicates with java part through java jni interface and uses corresponding native 7-Zip interface to perform needed operations.

First steps

Print count of the items in archive: 
ISevenZipInArchive archive 
        = SevenZip.openInArchive(ArchiveFormat.SEVEN_ZIP,
                new RandomAccessFileInStream(new RandomAccessFile(
                new File("archive.7z"), "r")));
System.out.println("Archive contains " 
        + archive.getNumberOfItems() + " items :-)");

Print list of the archived files using standard interface: 
ISevenZipInArchive archive
        = SevenZip.openInArchive(ArchiveFormat.SEVEN_ZIP,
                new RandomAccessFileInStream(new RandomAccessFile(
                new File("archive.7z"), "r")));
for (int index = 0; index < archive.getNumberOfItems(); index++) {
    System.out.println(
            "Filename: '" 
            + archive.getProperty(index, PropID.PATH)
            + "', packed size: " 
            + archive.getProperty(index, PropID.PACKED_SIZE));
}

Print list of the archived files using simple interface: 
ISevenZipInArchive archive 
        = SevenZip.openInArchive(ArchiveFormat.SEVEN_ZIP,
                new RandomAccessFileInStream(new RandomAccessFile(
                new File("archive.7z"), "r")));
ISimpleInArchive simpleInArchive = archive.getSimpleInterface();
for (ISimpleInArchiveItem inItem : simpleInArchive.getArchiveItems()) {
    System.out.println("Filename: '" + inItem.getPath()
            + "', packed size: " + inItem.getPackedSize());
}

Extract first item of the archive and print it: 
ISevenZipInArchive archive 
        = SevenZip.openInArchive(ArchiveFormat.SEVEN_ZIP,
                new RandomAccessFileInStream(new RandomAccessFile(
                new File("archive.7z"), "r")));
final StringBuilder stringBuilder = new StringBuilder();
archive.extractSlow(0, new ISequentialOutStream() {
    @Override
    public int write(byte[] data) {
        stringBuilder.append(new String(data));
        return data.length; // Amount of processed data
    }
});
System.out.println(stringBuilder.toString());
