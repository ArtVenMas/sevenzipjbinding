What's this all about?

7-Zip-JBinding is a java wrapper for 7-Zip C++ library. It allows extraction of many archive formats using very fast native library directly from java through JNI. Featues:

  • Extraction of
    7-Zip Zip Rar Tar Split Lzma Iso HFS GZip CPIO BZip2 Z Arj
  • It's cross-platform. Binaries are available for
    › MS-Windows › Linux 32/64 › Darwin Mac OS X
  • 12 example code snippets
  • Over 3000 JUnit tests
  • Planned: Creation of 7-Zip/Zip archives
  • Planned: Extraction of more archive formats, like RPM, CDEB, NSIS

How does it work?

The 7-Zip-JBinding consists of two parts: the java part and the native part. The java part introduces a cross-platform java interface of the 7-Zip library. The native part of 7-Zip-JBinding communicates with java part through java jni interface and uses corresponding native 7-Zip interface to perform operations.
