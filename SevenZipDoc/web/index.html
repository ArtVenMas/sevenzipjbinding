First Steps

Print count of items in archive: 
IInArchive archive = SevenZip.openInArchive(ArchiveFormat.SEVEN_ZIP,
	new InStreamImpl(new RandomAccessFile(new File("archive.7z"), "r")));

System.out.println("Archive contains "
		+ archive.getNumberOfItems() + " items :-)");
